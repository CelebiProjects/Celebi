Workflow Visualization & Provenance
================================

Commands for inspecting workflow structure and preserving results.

* ``draw-dag-graphviz`` – Generate a static DAG visualization
* ``draw-live-dag`` – Generate a live DAG with task status
* ``impress`` – Record an important result or state
* ``impression`` – List or retrieve impressions
* ``clean-impressions`` – Remove old impressions
* ``view`` – Open a file in the default viewer
